# ДЗ: Приложение для запоминания иностранных слов

В этом приложении пользователь сможет добавлять слова для изучения, проходить тесты для запоминания слов.
Это Single Page Application состоит из 3 страниц:
- Последние добавленные слова (Recently Added).
- Упражнениями (Go).
- Настройки (Settings).
На главном экране, на странице Recently Added пользователь видит список последних добавленных слов, может добавить новое слово в словарь.

На странице упражнений пользователь занимается тестированием своих знаний. Ему показывается слово на одном языке, и он должен написать его перевод на другой язык. Если перевод правильный, слово засчитывается, иначе показываем ошибку. Мы начнем с двух языков - русского и английского, будем расширять возможности приложения по мере написания программы.

На странице настроек пользователь выбирает языки, количество слов в упражнении, отводимое на упражнение время.

Навигация по страницам происходит с помощью ссылок в верхней части страниц, каждой странице соответствует отдельный url.

### Задание

Создать сервисы для работы с текстом
- Сервис перевода слова - должен запрашивать перевод через API (например, https://tech.yandex.com/translate/)
- Сервис хранения словаря - небольшая обертка для управления словарем с помощью `localStorage`
- Сервис добавления слов - должен разбивать текст на отдельные слова, запрашивать их перевод и сохранять в словарь для приложения.

Сервисы должны общаться с помощью библиотеки `RxJS`.

### Реализация

1. В разработанное [ранее](https://github.com/irina-dav/davydenko-irina-otus/tree/angular-01/angular/words-trainer) приложение добавлены **сервисы**:
    - `LocalStorage`: сервис для работы с  локальным хранилищем браузера.
    - `VocabularyStorage`: сервис для работы со словарём, использует сервис `LocalStorage`. Компоненты  работают с `VocabularyStorage`, не подключаясь напрямую к `LocalStorage`, таким образом, можно достаточно просто изменить способ хранения данных с `localStorage` на другой, вносить изменения в код компонентов не потребуется.
Аналогичным образом планируется реализовать сервисы для хранения настроек и результатов тренировок.
    - `VocabularyService`: сервис с разными методами для работы со словарём (перевод слова, перевод фразы и др).
    - `TranslatorService`: сервис для работы с API. Используется [MyMemory](https://mymemory.translated.net/doc/spec.php). К сожалению, с недавнего времени Яндекс.Переводчик приостановил выдачу [free ключей](https://translate.yandex.ru/developers/keys). Возможно, MyMemory даёт не самый лучший результат, но приемлемый для данной задачи.
    - `ViewService`: сервис c общей для разных компонентов функуиональностью. Пока это отображение уведомлений при помощи [ngx-toastr](https://www.npmjs.com/package/ngx-toastr).
2. Для взаимодействия с сервисами используется `RxJS`.
3. Добавлены features для вкладки **Recently Added**:
    - функциональность перевода фразы с разбивкой на отдельные слова, после чего можно исключить ненужные сочетания и сохранить оставшиеся в словарь,
    - возможность удаления ненужных пар (слово-перевод) из словаря.
4. Внесены минорные изменения в UI интерфейс/вёрстку. 
5. Страницы **Settings** и **Trainings** практически не менялись, запланированы в следующем ДЗ.
